Voice EPUB Reader

Welcome

This is an eBook reader app with a focus on reading aloud using text-to-speech software (TTS).

To start reading, tap 📁 button on bottom left and select an .epub book file downloaded on your device.

Controls

▶ — Start/stop reading aloud
⏪ — Previous sentence
⏩ — Next sentence
📁 — Open file
⚙️ — Options: voice, reading speed, etc.
Tap on right side of the screen — Next page
Tap on left side of the screen — Previous page
Pinch gesture — Adjust font size
